Welcome to IonQ’s package for hybrid neural networks!¶

This implements the framework for quantum-classical neural networks needed to complete IonQ’s QML challenge, hosted by SKKU.

The pages on this site document the package’s usage, for your reference.

Package library

  • Quantum-classical PyTorch modules
    • Classifier
    • Quantum module
    • Trainable module
  • Variational Ansatze
    • Base class for variational quantum circuits
    • Ansatz library
  • Qubit observables
    • Hamiltonian energy with respect to a variational ansatz
  • Quantum Functions
    • Quantum function category
    • Hybrid functions
    • PyTorch wrappers
  • Some useful helper methods
    • change_of_basis_circuit()
    • cvar_moments()
    • fast_ising_energies()
    • most_likely_states()
    • plot_cost_landscape()
    • robust_backend_call()
